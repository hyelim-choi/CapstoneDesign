# 기억담아란?

<img src="https://github.com/user-attachments/assets/c3bb15c9-9e88-4d51-a7cd-7ca7582772fd" width="30%" height="30%"/>


기억담아는 가족과의 소중한 "기억"을 "담는다"라는 의미를 가진 가족 커뮤니티입니다. <br/>
바쁜 현대사회 속 가족과 더 가까워질 수 있는 방법, `기억담아`를 찾아주세요.
